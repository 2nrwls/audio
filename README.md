# Audio BBcode

A [Flarum](http://flarum.org) extension. Adds an [audio] BBcode. This is compatible with my SM2 extension; feel free to install and use both.

### Installation

```sh
composer require zerosonesfun/flarum-bbcode-audio
```

### Updating

```sh
composer update zerosonesfun/flarum-bbcode-audio
php flarum cache:clear
```
